# Sharira Wellness Marketplace

Sharira is a premium wellness marketplace focused on Ayurveda and holistic health. This application helps users discover NABH/AYUSH-accredited treatment centres in India, match with practitioners, and manage their wellness journey with an AI-powered onboarding flow.

## 🚀 Experience the App

Sharira offers a seamless, premium onboarding experience that identifies your Ayurvedic Dosha (Vata, Pitta, Kapha) and matches you with relevant treatments like Panchakarma, Yoga Therapy, and Shirodhara.

### Key Features
- **AI-Powered Onboarding**: Personalised wellness profiling based on health goals and Ayurvedic principles.
- **Provider Marketplace**: Discover and book appointments at verified wellness villages and clinical hospitals.
- **Wellness Chat**: Direct communication for health consultations.
- **Health Records**: Centralised management of your medical history and treatment plans.

## 🛠️ Tech Stack

- **Framework**: React 18 with Vite
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Custom premium design system)
- **Icons**: Lucide React
- **Routing**: React Router DOM v6

## 🏃 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/pages/onboarding/`: The core multi-step onboarding journey.
- `src/components/`: Reusable UI components and layouts.
- `src/assets/`: Brand assets and images.
- `src/App.tsx`: Central routing and layout management.
